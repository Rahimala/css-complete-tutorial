The white-space CSS property specifies how white space inside an element is handled. It determines how consecutive whitespace characters are collapsed into a single space or preserved as multiple spaces. Possible values include: normal: Collapses whitespace and line breaks. This is the default value. nowrap: Prevents wrapping of text onto multiple lines, causing all text to appear on a single line. pre: Preserves all whitespace characters, including line breaks. pre-wrap: Preserves whitespace but allows line wrapping. pre-line: Collapses whitespace, but preserves line breaks.
