Flex Grow, Flex Shrink and Flex Basis

Flex Basis

Before we start growing or shrinking the elements, we need to should set the basis of the elements

The flex-basis CSS property specifies the flex basis which is the initial main size of a flex item.

This defines the default size of an element before the remaining space is distributed. It can be a length (e.g. 20%, 5rem, etc.) or a keyword.

Flex basis is similar to min-width. Try applying flex-wrap: wrap; on the flex containers.

  • flex-basis: 100px;
  • flex-basis: 200px;
  • flex-basis: 300px;

Flex Shrinks

What do we do if we don't have space?

The flex-shrink CSS property specifies the flex shrink factor of a flex item. Flex items will shrink to fill the container according to the flex-shrink number, when the default width of flex items is wider than the flex container.

If the parent has flex-wrap: wrap; flex-shrink won't work;

No Shrink

  • flex-shrink: 0;
    flex-basis: 200px;
  • flex-shrink: 0;
    flex-basis: 200px;

    flex-basis: 200px;
