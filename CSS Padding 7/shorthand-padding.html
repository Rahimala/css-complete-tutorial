If padding has 1 value

padding: 5px; means: padding-top:5px;padding-right:5px;padding-bottom:5px;padding-left:5px;

If padding has 2 values

padding: 20px 10px; means: padding-top:20px;padding-right:10px;padding-bottom:20px;padding-left:10px;

If padding has 3 values

padding: 30px 10px 20px; means: padding-top:30px;padding-right:10px;padding-bottom:20px;padding-left:10px;

If padding has 4 values

padding: 0 15px 20px 5px; means: padding-top:0;padding-right:15px;padding-bottom:20px;padding-left:5px;
