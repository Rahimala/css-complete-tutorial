In this example, the header has a position value of fixed, which means it stays at the top of the viewport and does not move when the page is scrolled. The sidebar has a position value of absolute, which means it is positioned relative to the nearest positioned ancestor (the header, in this case) and does not move when the page is scrolled. The main content has a margin applied that creates a gap to the left of 200px to accommodate the sidebar..
